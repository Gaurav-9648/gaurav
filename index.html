my project

Libary Management System

The mini-project “Library management system project in C” is a console application using the C programming language. This project compiled in Code Blocks with the GCC compiler. In this console application, you can do basic library management task like adding the book, view the added book, search the books, ..etc. This application based on file handling in C, where I have used a file-related function like fopen, fread, fwrite, ....etc. Good thing is that “Library management system project” is password-protected, so only authorized person able to login in this application. Also to increase the redabilty I have broken the application in different function. Each function of the project extensively use the file handing function, so it is also a great project to understand file handling.
